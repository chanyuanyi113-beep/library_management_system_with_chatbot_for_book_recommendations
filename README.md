Steps to access LMS
1. Download XAMPP version 8.2.12.
2. Open File Explorer.
3. Go to Windows (C:), open "xampp" folder, then open "htdocs" folder.
4. Copy "lms" folder provided in OneDrive and paste into "htdocs" folder.
5. Open XAMPP.
6. Start Apache and MySQL.
7. Click "Admin" button of MySQL.
8. Click "SQL" button at the top bar.
9. Copy all the content in lms_db in "lms" folder and paste into SQL and click "Go".
10. Open a new tab and paste "http://localhost/lms/index.php".
