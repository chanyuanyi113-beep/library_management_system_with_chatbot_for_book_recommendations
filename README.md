Steps to access LMS
1. Download XAMPP version 8.2.12.
2. Download all the codings and zipped files (unzipped them) and then save all in a folder named "lms".
3. Open File Explorer.
4. Go to Windows (C:), open "xampp" folder, then open "htdocs" folder.
5. Copy newly created "lms" folder and paste into "htdocs" folder.
6. Open XAMPP.
7. Start Apache and MySQL.
8. Click "Admin" button of MySQL.
9. Click "SQL" button at the top bar.
10. Copy all the content in "lms_db" and paste into SQL and click "Go".
11. Open a new tab and paste "http://localhost/lms/index.php".
